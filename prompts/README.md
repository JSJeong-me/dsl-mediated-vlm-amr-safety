# AMR Runtime Strategy Prompts

This folder contains three prompt templates used to define the runtime behavior of the Track A comparison study:

1. direct_execution_baseline_prompt.txt
2. schema_only_baseline_prompt.txt
3. dsl_guided_runtime_prompt.txt

The prompts correspond to the three compared runtime methods:
- Direct Execution Baseline
- Schema Only Baseline
- DSL Guided Proposed Runtime

The DSL Guided Proposed Runtime prompt defines the safety optimal runtime decision logic based on schema validation, semantic validation, and safety guard enforcement.
